=================
Integration guide
=================

This chapter describe frePPLe's capabilities to integrate with 
external systems.

The first sections describe the different ways to integrate your data
sources with frePPLe.

The batch command section contains information on how you can automate and
schedule tasks in frePPLe.

A final section contains information for people that require a low level
interaction with the frePPLe planning engine.
